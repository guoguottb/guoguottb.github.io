vue3

Guo TongBiao 发布于 2022-05-22

  • 🏷️ 知识点

1. vue3动机和优势

  1. API设计
  • vue3：compositionAPI，组合式API，功能逻辑写一起，方便逻辑复用对于维护和阅读更友好
  • vue2：optionsAPI，选项式API：data/metthods/computed（选项），优点：新手容易上手，功能：分散在各个选项中，从而维护，阅读，逻辑的复用也不友好（mixins）
  1. diif算法，vue3性能提高了很多，具体体现在给动态元素打上标记，对比虚拟dom的时候选择性对比
  2. vue3对ts支持更友好，因为底层是用ts重构的
  3. 响应式原理，vue2响应式原理是通过Object.defineproperty，vue3是用的Proxy

02. vite

  1. 前端构建工具，之间加载资源，遇到了模块化improt => ESM
  2. webpack，从入口开始打包，应用显示使用，创建项目通过：npm create vite

03. createApp

  1. 创建一个应用实例，参数就是一个组件对象
  2. 实例mount，参数是选择器，innerHTML

04. compositionAPI

  1. 入口函数：setup，执行机制在beforeCreate之前，里面拿不到this组件实例，return{ }，默认定义的数据，不是响应式
  2. 定义数据响应式，reactive：复杂数据，ref：简单/复杂都可以，vue3.2.0之后很少用reactive，推荐使用ref，性能提升200%+
  3. scrpt setup：相当于代码直接写在setup入口函数中，无需return，vue3.2.0才支持
  4. computed，本质是一个函数，需要引进来调用即可参数是函数（回调函数），进行逻辑的计算，并且返回值
  5. computed函数返回值就可以获取到
  6. 参数：还可以是一个对象，get：计算属性读的数据，set，当要修改计算属性的时候会触发改函数

05. watch

  • 参数一，监听一个数据，直接写对应的数据，监听多个数据用 [ ]，监听对象中的某个属性用函数 return 将其返回
  • 参数二，回调函数，当所监视的数据发生变化时执行的回调函数
  • 参数三，是一个对象，监视配置，例如 deep 深度监听 ，首次监听
  • 本质是一个函数，需要组合式API引入watch函数，其函数有三个参数

06. 生命周期函数

  • beforeCreate，created => setup（一个选项）
  • beforeMount，mounted，beforeUpdata,updated => 前面加一个on，本质是个函数
  • beforeDestroy，destroyed ，onBeforeUnMount => onUnMounted

07. 组件通信

  1. 父向子，通过属性绑定，子组件通过defineProps，
  2. 子向父，父组件 @定义自定义事件，子组件 defineEmite([ 约定事件名 ])，通过函数回调执行 emit（”事件名”，所传参数）
  3. 跨组件（组件间必须有联系，例如给子孙组件传值），传数据的组件通过 Provide（”名字”，数据）传递参数，后代组件通过 inject（”名字”）返回值获取到数据
  4. ref，ref可以将数据转化为响应式的，同时也可以获取到dom或者组件，首先通过ref创建个空响应数据，在给dom/组件绑定上ref等于定义的变量名
  5. 想要拿到组件内部的数据和方法，可以通过ref绑定组件

08. vue3是废弃到了过滤器

  • 所使用过滤器的话就是在js标签内定义一个方法，然后通过传参的方式将成功过滤的据 return 返回，让在模板中通过插值表达式直接使用函数传参即可

09. vue-router

  • 创建路由实例是通过 import { createRouter } fom “vue-router”，中的 createRouter函数内部配置对象，该函数有个返回值 router，挂载到 vue实例上即可，路由模式配置，history：createWebHashHistory() || createWebHistory()
  • 获取路由信息，当前的路由信息，在组件中 useRoute => import { useRoute } from “vue-router”，const route = userRoute()，路由实例，useRouter => import { useRouter } from “vue-router”，const router = useRouter()
